# Cycle Examples

## Writing Cycle

**Trigger:** Topic idea
**Done:** Published post with social snippets

```
Outline → Draft → Self-critique → Revise → SEO → Publish
```

Gates: Outline approval, final review before publish.

## Code Review Cycle

**Trigger:** PR opened
**Done:** Approved and merged

```
Understand → Test locally → Review → Comment → Re-review → Approve
```

Gates: At least one approval required.

## Research Cycle

**Trigger:** Question or topic
**Done:** Synthesized report with sources

```
Scope → Search → Filter → Analyze → Synthesize → Format
```

Gates: Scope confirmation, source quality check.
