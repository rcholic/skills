# Calculations -- Interior Design

## Material Quantities

**Flooring:** `Area = L x W`, add 10% waste (15% for diagonal). Boxes = area / m^2/box, round UP.

**Wall tile:** `Perimeter x tile height - doors/windows`, add 10% waste.

**Paint:** ~10-12 m^2/liter, 2 coats standard (3 if dark over light).

## Furniture Fit Check

**Door clearance:** `Diagonal = sqrt(width^2 + height^2)` must be < door width - 5cm

| Item | Size | Check |
|------|------|-------|
| 3-seat sofa | 220-250cm | Entry door, stairs turn |
| King bed | 200x200cm | Bedroom door turn radius |
| Dining table 6 | 180x90cm | +60cm each side for chairs |

## Circulation Minimums
