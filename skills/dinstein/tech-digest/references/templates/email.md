## Email-Specific Features

- **Rich subject line**: Includes date and article count
- **Executive summary**: High-level stats at the top
- **Full metadata**: Source attribution, timestamps, quality scores
- **Snippets**: Article summaries where available (Twitter, web search)
- **Technical section**: For ops teams monitoring system health
- **Top articles**: Highlight highest-quality content
- **Archive info**: Links to stored copies
- **HTML support**: Most email clients support markdown rendering

## Variables

### Basic Info
- `{{DATE}}` - Report date (February 15, 2026)
- `{{timestamp}}` - Generation timestamp
- `{{next_digest_time}}` - Next scheduled digest

### Summary Stats
- `{{total_sources}}` - Total number of sources
- `{{rss_sources}}` - Number of RSS sources
- `{{twitter_sources}}` - Number of Twitter sources  
- `{{web_sources}}` - Number of web search topics
- `{{total_articles}}` - Total articles in digest
- `{{topics_count}}` - Number of topic categories
- `{{hours}}` - Time window in hours

### Technical Stats
- `{{rss_feeds_ok}}` / `{{rss_feeds_total}}` - RSS success rate
- `{{twitter_accounts_ok}}` / `{{twitter_accounts_total}}` - Twitter success rate
- `{{web_topics_ok}}` / `{{web_topics_total}}` - Web search success rate
- `{{original_count}}` / `{{final_count}}` - Deduplication stats
- `{{multi_source_count}}` - Multi-source article count

### Article Data
- `{{topics}}` - Array of topic objects
- `{{emoji}}` / `{{label}}` / `{{description}}` - Topic metadata
- `{{articles}}` - Array of article objects per topic
- `{{title}}` / `{{link}}` - Article title and URL
- `{{source_name}}` - Source name
- `{{priority}}` - Boolean, true if high-priority source
- `{{date_formatted}}` - Human-readable date
- `{{snippet}}` - Article summary/description (when available)
- `{{topics_joined}}` - Comma-separated topic list
- `{{quality_score}}` - Article quality score
- `{{multi_source}}` / `{{source_count}}` - Multi-source indicators

### Archive
- `{{archive_path}}` - Path to archived digest file
