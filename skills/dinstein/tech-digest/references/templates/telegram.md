## Variables

- `{{DATE}}` - Report date (YYYY-MM-DD format)
- `{{topics}}` - Array of topic objects
- `{{emoji}}` - Topic emoji
- `{{label}}` - Topic display name
- `{{articles}}` - Array of article objects per topic
- `{{title}}` - Article title
- `{{link}}` - Article URL
- `{{summary}}` - Brief one-line summary (optional)
- `{{multi_source}}` - Boolean, true if from multiple sources
- `{{source_count}}` - Number of sources
- `{{total_sources}}` - Total sources used
- `{{total_articles}}` - Total articles in digest

## Message Splitting Strategy

Telegram has a 4096 character limit per message. If the digest exceeds this:

1. **First message**: Header + first N topics that fit
2. **Subsequent messages**: Remaining topics, each prefixed with `(cont.)`
3. **Last message**: Fixed sections (KOL, Trending, Blog Picks) + footer

Split at topic boundaries to keep sections intact.
