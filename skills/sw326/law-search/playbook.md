# 법률 검색 플레이북 — 개인 설정

## 관심 분야
- 근로법 (퇴직금, 해고, 연차)
- 임대차 (전세, 보증금, 계약)
- 개인정보보호 (동의, 처리, 유출)
- 관세/통상 (수출입, 관세율, 보복관세)
- 부동산 (매매, 등기, 세금)
- 일상 법률 (교통사고, 소비자보호, 층간소음)

## 출력 선호
- 조문 원문 + 쉬운 해설 병행
- 판례는 항상 1-2개 포함
- 실용적 조언 (구체적으로 뭘 할 수 있는지)
- 한국어 (법률 용어는 괄호 안에 쉬운 설명 추가)

## 검색 우선순위
1. 현행법령 (가장 최신)
2. 판례 (실제 적용 사례)
3. 법령해석례 (공식 해석)
4. 생활법령 (쉬운 해설)

## 출력 시 주의사항
- 폐지/개정된 법은 명확히 표시
- 시행일 미래인 법은 "아직 시행 전" 표시
- 민감한 사안은 "변호사 상담 추천" 명시
