# 출처 신뢰도 티어

> 공통 Tier 구조 + 인용 형식: [`../../common/source_tiers_base.md`](../../common/source_tiers_base.md)

## Tier 1 — 최고 신뢰도 (공식)
- **질병관리청** (kdca.go.kr) / **국가건강정보포털** (health.kdca.go.kr) — 질환·증상 정보
- **식품의약품안전처** (mfds.go.kr) / **e약은요** — 의약품 공식 정보
- **건강보험심사평가원** (hira.or.kr) — 병원 정보, 의료 통계
- **국민건강보험공단** (nhis.or.kr) — 건강검진, 보험 기준
- **대한의학회** / 각 전문학회 (대한내과학회, 대한외과학회 등)

## Tier 2 — 높은 신뢰도 (전문 매체)
- **서울대학교병원** / **세브란스병원** 건강정보 — 의학 콘텐츠
- **대한의사협회** — 의료 지침
- **헬스조선** / **메디컬투데이** — 의학 전문 매체

## Tier 3 — 보강용
- 각 병원 건강정보 블로그 (서울아산병원, 삼성서울병원 등 대형 병원)
- 정신건강 위기상담: **1393** (자살예방), **1577-0199** (정신건강 위기상담)

## 금지 출처
- 개인 블로그·카페 건강 정보 → 인용 금지
- 건강 유튜브·인플루언서 → 인용 금지
- 건강기능식품 광고성 콘텐츠 → 인용 금지

## 인용 형식
```markdown
# API
(국가건강정보포털 API: query=[질환명])
(식약처 의약품개요정보 API: 약품명=[약명])

# 웹 출처
[국가건강정보포털](https://health.kdca.go.kr) — 질환명

# 추정
"일반적으로 ~일 수 있습니다" + 진료 권고 병기
진단 단정 표현 절대 금지
```

## ⚠️ 의료 정보 특별 규칙
- **진단 단정 금지**: "~입니다" ❌ → "~일 수 있습니다 / ~가 의심됩니다" ✅
- **처방 금지**: 약 복용량·기간 구체 지정 금지
- **면책 문구**: 모든 응답 하단 필수
- **응급 판단 보수적**: 불확실하면 무조건 119 안내
