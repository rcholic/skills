# 스크립트 디렉토리

## 상태

| 스크립트 | API | 상태 | 비고 |
|---------|-----|------|------|
| `kdca_health.sh` | 국가건강정보포털 | ⏳ 키 승인 대기 | health.kdca.go.kr 신청 완료 (2026-02-22) |
| `drug_info.sh` | 식약처 의약품개요정보 | ⏳ 키 승인 대기 | data.go.kr/15075057 활용신청 완료 |

## 키 승인 후 할 일

1. 국가건강정보포털 키 발급 → `~/.config/kdca/api_key` 저장
2. `kdca_health.sh` 스크립트 구현
3. 식약처 키 활성화 확인 → `drug_info.sh` 스크립트 구현
