# Collaboration Helper reference

## Task lifecycles
- `open`: newly created requests that need attention.
- `in-progress`: someone is working on the task and checking in regularly.
- `done`: completed work.

## Priority levels

## Notes

- Tag relevant teammates in the note when you need cross-agent help.
