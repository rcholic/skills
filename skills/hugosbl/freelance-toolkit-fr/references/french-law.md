# Mentions légales obligatoires — Factures en France

## Mentions obligatoires (Article L441-9 du Code de commerce)

1. **Date d'émission** de la facture
2. **Numérotation** : séquentielle et sans rupture (format recommandé : YYYY-NNN)
3. **Identité du vendeur** : nom/raison sociale, adresse, SIRET, code APE/NAF
4. **Identité de l'acheteur** : nom/raison sociale, adresse, SIRET (si professionnel)
5. **Désignation** des produits/services : nature, quantité, prix unitaire HT
6. **Montant total HT** et **TTC**
7. **Taux de TVA** applicable (ou mention d'exonération)
8. **Date de la prestation** ou période concernée
9. **Date d'échéance** du paiement
10. **Conditions d'escompte** (ou mention « Pas d'escompte pour paiement anticipé »)
11. **Pénalités de retard** : taux applicable (minimum 3× le taux d'intérêt légal)
12. **Indemnité forfaitaire** pour frais de recouvrement : **40 €**

## Micro-entreprise / Auto-entrepreneur

Si le prestataire est en franchise en base de TVA :

> « TVA non applicable, article 293 B du Code général des impôts »

- Pas de TVA à facturer
- Pas de ligne TVA sur la facture
- Le total HT = total TTC

## Mentions complémentaires recommandées

- Numéro de bon de commande (si applicable)
- Conditions de paiement (virement, chèque, délai)
- RIB / IBAN pour le paiement
- Mention « Membre d'une association de gestion agréée, le règlement par chèque et carte bancaire est accepté » (si applicable)

## Pénalités de retard

Taux minimum : 3 × taux d'intérêt légal en vigueur (ou taux BCE + 10 points).
Indemnité forfaitaire de recouvrement : **40 €** (Article D441-5 du Code de commerce).

## Conservation

Les factures doivent être conservées **10 ans** (Article L123-22 du Code de commerce).
