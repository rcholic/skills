# Network Map (Etherscan V2)

Unified API base URL:
- `https://api.etherscan.io/v2/api`

Use `chainid` to select network:

| Network | chainid | Explorer URL |
| --- | ---: | --- |
| Ethereum mainnet | `1` | `https://etherscan.io/` |
| Ethereum hoodi | `560048` | `https://hoodi.etherscan.io/` |
| Taiko mainnet | `167000` | `https://taikoscan.io/` |
| Taiko hoodi | `167013` | `https://hoodi.taikoscan.io/` |

Notes:
- Etherscan V2 supports multichain access through one API key.
- Contract ABI/source endpoints are available on supported chains.
- Refresh chain support reference: `https://docs.etherscan.io/introduction`
- Full supported list reference: `https://docs.etherscan.io/supported-chains`
