# Гайд по созданию приложения Pinterest

Чтобы использовать этот навык, тебе нужно создать приложение на портале для разработчиков Pinterest.

## Шаг 1: Регистрация
1. Перейди на [Pinterest Developers](https://developers.pinterest.com/).
2. Войди под своим аккаунтом Pinterest.
3. Прими условия использования (Terms of Service), если это твой первый раз.

## Шаг 2: Создание приложения
1. Нажми на кнопку **"Create app"**.
2. Введи название приложения (например, "My OpenClaw Assistant") и описание.
3. Нажми **"Create"**.

## Шаг 3: Получение учетных данных
1. В настройках приложения ты увидишь **App ID** и **App Secret**. Сохрани их, они понадобятся для OAuth.
2. В разделе **"Redirect URIs"** добавь `http://localhost:8080` (или другой адрес, если ты настраиваешь сервер). Для локального скрипта `auth.py` используй `http://localhost:8080`.

## Шаг 4: Настройка разрешений (Scopes)
Pinterest API v5 использует области видимости. Для полной работы тебе понадобятся:
- `boards:read`
- `boards:write`
- `pins:read`
- `pins:write`
- `user_accounts:read`
- `ads:read` (для аналитики)

Теперь ты готов запустить скрипт `scripts/auth.py` для получения токена доступа.
