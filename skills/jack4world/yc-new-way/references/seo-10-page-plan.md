# SEO 10-Page Plan (ship in 2–3 days)

## Page types
1) Money pages (2)
- Model/Category pages with intent (e.g., “Ultra 1 二手价格/避坑”)

2) Decision pages (3)
- Comparisons (“A vs B 怎么选”)
- Buy-new vs used threshold page

3) Utility pages (5)
- Checklists (“10分钟验机清单”)
- Scam/avoid pages
- Glossary (“官换/监管机/资源机”)

## Content rules
- Evidence-first: cite sources, link out.
- Fixed layout: TL;DR box → price/criteria → checklist → alternatives.
- Internal links: every page links to 2 money pages.

## Minimal tech
- Any static site/CMS is fine.
- Generate pages from a simple JSON/Markdown repo later.
